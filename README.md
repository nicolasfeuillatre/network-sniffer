# network-sniffer
A simple Python network sniffer that captures and displays key information such as IP addresses, protocols, ports, and packet summaries, using the Scapy library.

This project was built by following this article:
[Build a Network Sniffer in Python Using Scapy](https://pythonscripting.hashnode.dev/build-a-network-sniffer-in-python-using-scapy)
